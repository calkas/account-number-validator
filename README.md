# account-number-validator
Account number validator

## Running

First `tesseract-ocr` is needed 

> sudo apt-get install tesseract-ocr

> python3 main.py -i image.jpg -an "00 1234 1234 0000 1234 1234 1234"

## venv

env activation

> source activate

Python packages
> pip install -r requirements.txt
